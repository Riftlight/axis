# Axis
Axis is a project made in Godot for [onekey](https://onekey.hackclub.com/). You play as Axis, a character with the power to bend the direction of its gravity with just one keypress.
Axis is made in Godot with .NET (C#), as, unlike GDScript, i was familiar with the language when i started the project. Unfortunately, this means that it cannot be exported to web, and instead must be downloaded.
This is my first project made in Godot, and i'm sure i choose poor implementation paths at times (not one usage of signals to be found), but i'm pretty happy with what i was able to make.

# Controls
Place your mouse cursor relative to the character in the direction you want to go, and press Space to shift your gravity to that direction.
Press Esc to return to the main menu (i know, not one key, but i couldn't think of any better way!)
Hold Space while unmoving to restart a level. Remember, the level timer only starts after your first movement!
